Purpose

To prove that if a function $ f:\{0,1\}^N \Rightarrow \{0,1\}$ is hard-on-average in a weak sense then we can construct $f'$ which is hard-on-average in a strong sense. Definition : A function $ f:\{0,1\}^N \Rightarrow \{0,1\}$ is $ (S,\deta)$-hard iff for any circuit $ C$ with size $\leq S$ $$ Pr[C(x) \neq f(x)]
