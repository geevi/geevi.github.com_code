--- layout: post title: Some Theorem's in First Order Logic published: false meta: _edit_last: "5862988" tags: [] type: post status: draft --- Hintikka set (Definition) : Let L be some first order language. $ H \subset L$ is called a first-order Hintikka iff
  1. For any atomic formula $ \phi$, not both $ \phi$ and $ \not\phi \in H$.
  2. $ \top, \not\bottom \notin H$
  3. $ \alpha \in H \implies \alpha_{1} and \alpha_{2} \in H$
  4. $ \beta \in H \implies \beta_{1} and \beta_{2} \in H$
  5. $ \gamma \in H \implies \gamma(t)  \in H \mbox{for all closed terms}  \in Lt$
  6. $ \delta \in H \implies \delta(t)  \in H \mbox{for some closed term} t \in L$
Hintikka's Lemma : Any first-order hintikka set $ H$ is satisfiable in a Herbrand model.
